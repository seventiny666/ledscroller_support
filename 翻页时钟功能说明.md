# 翻页时钟功能说明

## 功能概述
在首页新增了一个翻页时钟卡片，点击后进入全屏翻页时钟效果，显示手机的真实时间。

## 实现细节

### 1. 新增文件
- `GlowLed/FlipClockViewController.swift` - 翻页时钟视图控制器

### 2. 翻页时钟特性
- **显示真实时间**：自动读取手机系统时间（小时:分钟）
- **翻页动画**：数字变化时有3D翻页效果
- **横屏显示**：全屏横屏模式
- **自动更新**：每秒检测时间变化并更新
- **4位数字**：小时十位、小时个位、分钟十位、分钟个位

### 3. 翻页动画效果
- 上半部分向下翻转（0.3秒）
- 下半部分从下向上翻转（0.3秒）
- 使用3D透视变换（CATransform3D）
- 翻转时有透明度渐变效果

### 4. 首页预览效果
- 显示当前真实时间的静态预览
- 4个翻页数字 + 冒号分隔符
- 深灰色背景，白色数字
- 上下两半分离的翻页样式

### 5. 数据模型更新
在 `LEDItem.swift` 中添加：
- `isFlipClock: Bool` - 标识是否为翻页时钟

### 6. 默认卡片
```swift
LEDItem(
    id: "flip-clock-special",
    text: "🕐 翻页时钟",
    fontSize: 45,
    textColor: "#FFFFFF",
    backgroundColor: "#0D0D0D",
    glowIntensity: 2.0,
    scrollType: .none,
    speed: 1.0,
    isFlipClock: true
)
```

### 7. 视觉设计
- **背景色**：深黑色 (#0D0D0D)
- **数字颜色**：白色
- **上半部分**：深灰色 (RGB: 0.25, 0.25, 0.25)
- **下半部分**：稍浅灰色 (RGB: 0.3, 0.3, 0.3)
- **分隔线**：黑色 (RGB: 0.1, 0.1, 0.1)
- **数字大小**：100pt（全屏）/ 24pt（预览）
- **圆角**：8px（全屏）/ 3px（预览）

### 8. 交互逻辑
- 点击首页翻页时钟卡片 → 进入全屏翻页时钟
- 不可编辑、不可删除（特殊卡片）
- 点击"完成"按钮返回首页

## 使用方法

### 在Xcode中添加文件
1. 打开Xcode项目
2. 在左侧项目导航器中，右键点击"GlowLed"文件夹
3. 选择"Add Files to GlowLed..."
4. 找到并选择 `GlowLed/FlipClockViewController.swift`
5. 确保勾选：
   - ✅ Copy items if needed
   - ✅ Create groups
   - ✅ Add to targets: GlowLed
6. 点击"Add"
7. 按 Cmd+B 构建项目

### 测试步骤
1. 运行应用
2. 在首页找到"🕐 翻页时钟"卡片
3. 点击卡片进入全屏翻页时钟
4. 观察时间是否正确显示
5. 等待1分钟，观察分钟数字是否有翻页动画
6. 点击"完成"按钮返回首页

## 技术实现

### FlipDigitView 类
- 自定义翻页数字视图
- 包含上半部分和下半部分
- 实现3D翻转动画
- 支持0-9数字显示

### 动画流程
1. 检测数字变化
2. 创建上半部分翻转视图
3. 执行上半部分向下翻转（90度）
4. 更新下半部分数字
5. 创建下半部分翻转视图
6. 执行下半部分向上翻转（从-90度到0度）
7. 更新上半部分数字

### Timer 定时器
- 每秒触发一次
- 检查时间变化
- 只更新变化的数字（优化性能）

## 注意事项
1. 翻页时钟卡片不能编辑或删除
2. 全屏显示时强制横屏
3. 返回首页时恢复竖屏
4. Timer在页面消失时自动停止
5. 时间格式为24小时制（00:00 - 23:59）
