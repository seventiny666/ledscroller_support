# 创作卡片背景图片显示修复说明

## 问题描述

用户在创建LED时选择了背景图片（霓虹灯看板、偶像应援、LED横幅），但保存后在"我的创作"面板上显示的卡片封面只显示文字，没有显示背景图片。

## 问题原因

1. `LEDItem`结构体中没有保存背景图片名称的属性
2. `LEDCreateViewController`中的`selectedBackgroundImage`变量只在预览时使用，保存时没有存储到`LEDItem`
3. `MyCreationsViewController`中的`CreationCell`只显示背景颜色，没有处理背景图片

## 修复方案

### 1. 在LEDItem中添加背景图片属性

**文件**: `GlowLed/LEDItem.swift`

```swift
// LED屏幕数据模型
struct LEDItem: Codable {
    var id: String
    var text: String
    var fontSize: CGFloat
    var textColor: String
    var backgroundColor: String
    var backgroundImageName: String? // ✅ 新增：背景图片名称
    var glowIntensity: CGFloat
    var scrollType: ScrollType
    var speed: CGFloat
    var fontName: String
    var isFireworks: Bool
    var isFireworksBloom: Bool
    var isFlipClock: Bool
    var isLoveRain: Bool
    var createdAt: Date
    
    // ...
}
```

**修改内容**:
- 添加`backgroundImageName: String?`属性，用于保存背景图片名称（如：neon_1, idol_2, led_3）
- 在`init`方法中添加`backgroundImageName`参数

### 2. 保存背景图片信息

**文件**: `GlowLed/LEDCreateViewController.swift`

#### 2.1 保存时存储背景图片

```swift
private func updateCurrentItem() {
    let text = textField.text ?? ""
    currentItem.text = (text == "led content" && textField.textColor == UIColor.white.withAlphaComponent(0.3)) ? "" : text
    
    let scrollTypes: [LEDItem.ScrollType] = [.none, .scrollLeft, .scrollRight, .scrollUp, .scrollDown]
    currentItem.scrollType = scrollTypes[scrollTypeSegment.selectedSegmentIndex]
    
    // ✅ 新增：保存背景图片名称
    currentItem.backgroundImageName = selectedBackgroundImage
}
```

#### 2.2 加载时恢复背景图片

```swift
private func loadCurrentData() {
    // ... 其他加载代码
    
    // ✅ 新增：加载背景图片
    selectedBackgroundImage = currentItem.backgroundImageName
    
    // ... 其他加载代码
}
```

### 3. 在创作卡片中显示背景图片

**文件**: `GlowLed/MyCreationsViewController.swift`

#### 3.1 添加背景图片视图

```swift
class CreationCell: UICollectionViewCell {
    
    private let previewView = UIView()
    private let backgroundImageView = UIImageView() // ✅ 新增：背景图片视图
    private let textLabel = UILabel()
    
    // ...
}
```

#### 3.2 设置背景图片布局

```swift
private func setupUI() {
    // ... 预览区域设置
    
    // ✅ 新增：背景图片视图
    backgroundImageView.contentMode = .scaleAspectFill
    backgroundImageView.clipsToBounds = true
    backgroundImageView.translatesAutoresizingMaskIntoConstraints = false
    previewView.addSubview(backgroundImageView)
    
    // ... 文字标签设置
    
    NSLayoutConstraint.activate([
        // ... 其他约束
        
        // ✅ 新增：背景图片约束
        backgroundImageView.topAnchor.constraint(equalTo: previewView.topAnchor),
        backgroundImageView.leadingAnchor.constraint(equalTo: previewView.leadingAnchor),
        backgroundImageView.trailingAnchor.constraint(equalTo: previewView.trailingAnchor),
        backgroundImageView.bottomAnchor.constraint(equalTo: previewView.bottomAnchor),
        
        // ... 其他约束
    ])
}
```

#### 3.3 配置背景图片显示

```swift
func configure(with item: LEDItem) {
    // ✅ 新增：更新背景（图片或颜色）
    if let imageName = item.backgroundImageName, let image = UIImage(named: imageName) {
        // 显示背景图片
        backgroundImageView.image = image
        backgroundImageView.isHidden = false
        previewView.backgroundColor = .clear
    } else {
        // 显示背景颜色
        backgroundImageView.image = nil
        backgroundImageView.isHidden = true
        previewView.backgroundColor = UIColor(hex: item.backgroundColor)
    }
    
    // 文字和霓虹效果
    textLabel.text = item.text
    textLabel.font = UIFont(name: item.fontName, size: 24) ?? .boldSystemFont(ofSize: 24)
    textLabel.textColor = UIColor(hex: item.textColor)
    
    textLabel.layer.shadowColor = UIColor(hex: item.textColor).cgColor
    textLabel.layer.shadowRadius = 8 * item.glowIntensity
    textLabel.layer.shadowOpacity = Float(item.glowIntensity * 0.3)
    textLabel.layer.shadowOffset = .zero
}
```

## 数据流程

### 创建/编辑LED
1. 用户在"背景"Tab选择背景图片（霓虹灯看板/偶像应援/LED横幅）
2. `selectedBackgroundImage`保存图片名称（如：neon_1）
3. 预览区域显示背景图片+文字
4. 点击"保存"
5. `updateCurrentItem()`将`selectedBackgroundImage`保存到`currentItem.backgroundImageName`
6. `LEDDataManager`保存到UserDefaults

### 显示创作卡片
1. `MyCreationsViewController`加载数据
2. `LEDDataManager`从UserDefaults读取数据
3. `CreationCell.configure(with:)`接收`LEDItem`
4. 检查`item.backgroundImageName`
5. 如果有图片名称，加载并显示图片
6. 如果没有图片名称，显示背景颜色
7. 文字叠加在背景上

## 兼容性处理

### 旧数据兼容
```swift
var backgroundImageName: String? // 可选类型
```
- 使用可选类型`String?`
- 旧数据没有此字段，解码时自动为`nil`
- 显示时会回退到背景颜色

### 显示逻辑
```swift
if let imageName = item.backgroundImageName, let image = UIImage(named: imageName) {
    // 有图片：显示图片
} else {
    // 无图片：显示颜色
}
```
