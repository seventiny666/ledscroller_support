# 两种独立烟花效果说明

## 实现时间
2026-03-01

## 功能概述
创建了两个独立的烟花卡片，每个卡片对应不同的烟花效果：

1. **🎆 烟花** - 从底部发射上升后爆炸
2. **🎇 烟花绽放** - 直接在点击位置绽放

## 两种烟花对比

| 特性 | 🎆 烟花 (第一种) | 🎇 烟花绽放 (第二种) |
|------|-----------------|-------------------|
| 文件 | FireworksViewController.swift | FireworksBloomViewController.swift |
| 卡片ID | fireworks-special | fireworks-bloom-special |
| 卡片文字 | 🎆 烟花 | 🎇 烟花绽放 |
| 文字颜色 | 金色 (#FFD700) | 粉色 (#FF69B4) |
| 发射方式 | 底部发射→上升→爆炸 | 直接在点击位置绽放 |
| 预览效果 | 从底部向上喷射 | 从中心向外绽放 |
| 循环方式 | 持续向上喷射 | 2秒循环绽放 |
| 烟花类型 | 5种随机类型 | 5种随机类型 |
| 颜色组合 | 5种配色 | 5种配色（偏粉紫系） |

### 1. 数据模型扩展
```swift
struct LEDItem: Codable {
    var isFireworks: Bool        // 第一种烟花
    var isFireworksBloom: Bool   // 第二种烟花绽放

```

### 2. 默认卡片配置
```swift

// 第二种：🎇 烟花绽放
LEDItem(

    isFireworks: true,
    isFireworksBloom: true

```

### 3. 点击逻辑区分
```swift
func collectionView(_ collectionView: UICollectionView, didSelectItemAt indexPath: IndexPath) {

    }
    
    // 普通LED卡片
    // ...

```

### 4. 卡片预览效果区分
```swift
func configure(with item: LEDItem) {
    if item.isFireworksBloom {
        setupFireworksBloomEffect()  // 中心绽放
    }
    else if item.isFireworks {
        setupFireworksEffect()       // 向上喷射
    }
    else {
        // 普通LED霓虹效果
    }
}
```

## 第一种烟花：🎆 烟花

### 特点
- 从屏幕底部随机位置发射
- 带上升轨迹动画
- 到达点击位置后爆炸
- 5种烟花类型随机

### 卡片预览
- 从卡片底部持续向上喷射
- 45度扇形向上
- 持续动画

### 全屏效果
```swift
private func launchFirework(at targetPosition: CGPoint) {
    // 从底部发射
    let startPosition = CGPoint(x: startX, y: view.bounds.height + 20)
    
    // 动画上升
    let animation = CABasicAnimation(keyPath: "emitterPosition")
    animation.fromValue = startPosition
    animation.toValue = targetPosition
    
    // 到达后爆炸
    DispatchQueue.main.asyncAfter(deadline: .now() + duration) {
        self.createFireworkExplosion(at: targetPosition)
    }
}
```

### 特点

- 5种烟花类型随机

- 360度全方向扩散

## 5种烟花类型

两种烟花都支持以下5种类型：

### 1. 爆裂型 (Burst)
- 360度均匀散开
- 速度快，扩散范围大

### 2. 柳树型 (Willow)
- 向下垂落
- 重力效果明显

### 3. 棕榈型 (Palm)
- 向上散开后下落
- 形成棕榈树状

### 4. 环形 (Ring)
- 水平扩散
- 形成圆环状

### 5. 菊花型 (Chrysanthemum)
- 密集绽放
- 粒子数量最多

## 性能优化

### 共同优化
- 硬件加速（CAEmitterLayer）
- 自动清理过期粒子
- weak引用避免循环
- 离开页面释放资源

### 第一种烟花
- 火箭粒子生命周期短（0.3秒）
- 爆炸后3秒清理

### 第二种烟花
- 直接爆炸，无上升阶段
- 3秒后清理
