## 如果看不到所有卡片

### 方案1：清除旧数据
在`AppDelegate.swift`的`application(_:didFinishLaunchingWithOptions:)`中添加：

```swift
// 清除旧数据（仅用于测试）
UserDefaults.standard.removeObject(forKey: "savedLEDItems")
```

然后重新运行应用。

### 方案2：添加调试代码
在`TemplateSquareViewController.swift`的`viewDidLoad`中添加：

```swift
override func viewDidLoad() {
    super.viewDidLoad()
    setupUI()
    
    // 调试代码
    let allItems = LEDDataManager.shared.loadItems()
    print("=== 所有卡片 ===")
    print("总数: \(allItems.count)")
    allItems.forEach { print("- \($0.text) (id: \($0.id))") }
    
    let otherItems = allItems.filter { 
        !$0.isFlipClock && 
        !$0.isNeonTemplate && 
        !$0.isIdolTemplate && 
        !$0.isLEDTemplate 
    }
    print("\n=== 其他分类 ===")
    print("总数: \(otherItems.count)")
    otherItems.forEach { print("- \($0.text)") }
}
```

查看控制台输出，确认数据加载正确。

### 方案3：检查ID
在`LEDItem.swift`的`getDefaultItems()`方法中，确认所有卡片都有明确的ID：


## 常见问题

### Q1: 只看到4个卡片
A: 清除UserDefaults数据，重新运行应用。

### Q2: 卡片重复
A: 正常现象。旧数据中的卡片会保留，新的预设卡片也会添加。

### Q3: 烟花效果不显示
A: 确认`LEDItem.swift`中的`isFireworks`和`isFireworksBloom`标识正确。

### Q4: 点击卡片没反应
A: 检查`TemplateSquareViewController.swift`中的`handleItemTap`方法。

### Q5: 卡片被遮挡
A: 已修复。Row高度350px，底部contentInset 20px。
