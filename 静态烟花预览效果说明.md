# 静态烟花预览效果说明

## 修改时间
2026-03-01

## 修改内容
将首页烟花卡片的预览效果从动画改为静态图案，两种烟花显示不同的静态效果。

## 两种静态预览效果

### 1. 🎆 烟花（第一种）- 向上发射轨迹
**视觉效果：**
- 从卡片底部向上的发射轨迹
- 15个粒子形成上升路径
- 顶部有8个粒子形成小爆炸
- 粒子颜色：黄、橙、红渐变
- 粒子逐渐变淡（模拟距离感）

**实现方式：**
```swift
private func setupStaticLaunchFirework() {
    // 创建向上的轨迹粒子
    for i in 0..<15 {
        let y = startY - CGFloat(i) * 6
        let xOffset = CGFloat.random(in: -8...8)
        let particle = createStaticParticle(color: colors[i % colors.count])
        particle.center = CGPoint(x: x, y: y)
        particle.alpha = 1.0 - (CGFloat(i) / 15.0) * 0.5
    }
    
    // 顶部小爆炸（8个粒子）
    for i in 0..<8 {
        let angle = (CGFloat(i) / 8.0) * .pi * 2
        let radius: CGFloat = 15
        // 360度均匀分布
    }
}
```

### 2. 🎇 烟花绽放（第二种）- 礼花炸开效果
**视觉效果：**
- 从卡片中心向外辐射
- 24个粒子360度均匀分布
- 形成完整的圆形爆炸图案
- 粒子颜色：粉、紫、品红、红、橙、白
- 半径40像素的圆形分布

**实现方式：**
```swift
private func setupStaticBloomFirework() {
    let particleCount = 24
    let radius: CGFloat = 40
    
    // 360度均匀分布
    for i in 0..<particleCount {
        let angle = (CGFloat(i) / CGFloat(particleCount)) * .pi * 2
        let x = centerX + cos(angle) * radius
        let y = centerY + sin(angle) * radius
        
        let particle = createStaticParticle(color: colors[i % colors.count])
        particle.center = CGPoint(x: x, y: y)
    }
}
```

## 静态粒子设计

### 粒子属性
```swift
private func createStaticParticle(color: UIColor) -> UIView {
    let size: CGFloat = 4
    let particle = UIView(frame: CGRect(x: 0, y: 0, width: size, height: size))
    particle.backgroundColor = color
    particle.layer.cornerRadius = size / 2  // 圆形
    
    // 发光效果
    particle.layer.shadowColor = color.cgColor
    particle.layer.shadowRadius = 3
    particle.layer.shadowOpacity = 0.8
    particle.layer.shadowOffset = .zero
    
    return particle
}
```

### 粒子特点
- 大小：4x4 像素
- 形状：圆形
- 发光：阴影半径3，透明度0.8
- 颜色：根据烟花类型不同

## 设计理念

### 为什么使用静态图案？

1. **性能最优**
   - 无动画计算
   - 无定时器
   - 无粒子系统开销

2. **视觉清晰**
   - 静态图案更容易识别
   - 一眼就能看出是烟花效果
   - 不会分散注意力

3. **电池友好**
   - 无持续动画
   - 无CPU占用
   - 延长电池寿命

4. **用户体验**
   - 列表滚动更流畅
   - 加载速度更快
   - 视觉效果清晰

### 静态 vs 动画对比

| 特性 | 静态图案 | 动画效果 |
|------|---------|---------|
| CPU占用 | 0% | 5-10% |
| 内存占用 | 极低 | 中等 |
| 滚动流畅度 | 完美 | 良好 |
| 视觉识别 | 清晰 | 需要观察 |
| 电池消耗 | 极低 | 中等 |
| 实现复杂度 | 简单 | 复杂 |
