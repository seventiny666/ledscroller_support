# 创建页面Tab重构说明

## 功能概述

将创建LED页面重构为Tab切换模式，将功能分为"字体"、"背景"、"动画"三个分类，提升用户体验和操作效率。

## 重构内容

### 1. Tab切换结构

在文字输入框下方添加Tab切换控件，包含三个Tab：
- **字体** - 字体相关设置
- **背景** - 背景相关设置
- **动画** - 动画相关设置

### 2. 字体Tab

包含以下功能：
- **字体大小** - 滑块调节（30 - 屏幕高度90%）
- **字体选择** - 4种字体（苹方、黑体、宋体、楷体）
- **文字颜色** - 2行14个纯色选择
- **渐变颜色** - 2行14个渐变选择
- **霓虹强度** - 滑块调节（0 - 10，从原来的0-5扩展）

### 3. 背景Tab

包含以下功能：
- **选择背景** - 纯色背景（6个）+ 纹理背景（6个）
- **霓虹灯看板** - 4个模版背景（支持图片或占位符）
- **偶像应援** - 4个模版背景（支持图片或占位符）
- **LED横幅** - 4个模版背景（支持图片或占位符）

### 4. 动画Tab

包含以下功能：
- **滚动方式** - 5种方式（静止、左滚、右滚、上滚、下滚）
- **播放速度** - 滑块调节（0.5 - 3.0）

## 技术实现

### 1. Tab切换控件

```swift
private let tabSegment = UISegmentedControl(items: ["字体", "背景", "动画"])
```

### 2. 三个Tab视图

```swift
private var fontTabView: UIView!        // 字体Tab
private var backgroundTabView: UIView!  // 背景Tab
private var animationTabView: UIView!   // 动画Tab
```

### 3. Tab切换逻辑

```swift
@objc private func tabChanged() {
    showTab(index: tabSegment.selectedSegmentIndex)
}

private func showTab(index: Int) {
    fontTabView.isHidden = (index != 0)
    backgroundTabView.isHidden = (index != 1)
    animationTabView.isHidden = (index != 2)
}
```

## 新增功能

### 1. 霓虹强度扩展

- **原来**: 0-5
- **现在**: 0-10
- **计算公式**:
  - 发光半径: `2.5 * glowIntensity` (0-25范围)
  - 发光透明度: `min(glowIntensity / 10.0, 1.0)` (0-1范围)

### 2. 模版背景

三种模版类型，每种4个选项：

#### 霓虹灯看板
- 占位符颜色: 紫、青、金、粉
- 图片命名: `neon_1` ~ `neon_4`
- Tag范围: 500-503

#### 偶像应援
- 占位符颜色: 粉红、粉紫、深粉、紫
- 图片命名: `idol_1` ~ `idol_4`
- Tag范围: 510-513

#### LED横幅
- 占位符颜色: 绿、浅绿、青、青绿
- 图片命名: `led_1` ~ `led_4`
- Tag范围: 520-523

## 用户操作流程

### 创建LED流程

1. **输入文字**
   - 在文字输入框输入LED文字内容
   - 实时预览显示效果

2. **设置字体**
   - 切换到"字体"Tab
   - 调整字体大小、选择字体
   - 选择文字颜色或渐变
   - 调整霓虹强度

3. **设置背景**
   - 切换到"背景"Tab
   - 选择纯色、纹理或模版背景
   - 预览区域实时显示背景效果

4. **设置动画**
   - 切换到"动画"Tab
   - 选择滚动方式
   - 调整播放速度

5. **保存**
   - 点击右上角"保存"按钮
   - LED效果保存到创作列表

## 设计规范

### 1. 颜色规范
- Tab选中色: #8EFFE6（青绿色）
- 标签文字色: RGB(0.6, 0.8, 1.0)（柔和蓝色）
- 按钮边框: 白色30%透明度（未选中）/ 白色100%（选中）

### 2. 间距规范
- Tab高度: 40px
- Tab与内容间距: 15px
- 标签与控件间距: 30px
- 控件之间间距: 50-70px
- 左右边距: 20px

### 3. 按钮规范
- 颜色按钮: 36×36px，圆形，间距12px
- 背景按钮: 48×36px，圆角6px，间距8px
- 模版按钮: 48×36px，圆角6px，间距8px

### 4. 交互规范
- Tab切换: 点击切换，平滑过渡
- 按钮选择: 边框加粗，白色高亮
- 滑块调节: 实时预览，流畅响应
- 互斥选择: 纯色/渐变、不同背景类型

### 2. 约束管理
- 每个Tab视图独立约束
- 使用相同的topAnchor（yOffset）
- 通过isHidden控制显示/隐藏
- 避免约束冲突

### 3. 状态管理
- selectedGradientIndex: 渐变选择索引
- selectedTextColorIndex: 文字颜色索引
- selectedBgColorIndex: 背景颜色索引
- 互斥选择通过清除其他选择实现
