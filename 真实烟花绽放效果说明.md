# 真实烟花绽放效果说明

## 实现时间
2026-03-01

## 功能概述
实现了真实的烟花绽放效果，包括：
1. 从底部发射上升
2. 到达顶点后爆炸绽放
3. 5种不同的烟花类型
4. 首页卡片循环播放烟花绽放动画

## 烟花效果分析

根据提供的烟花图片，实现了以下5种烟花类型：

### 1. 爆裂型 (Burst)
- 360度均匀散开
- 粒子速度快，扩散范围大
- 类似图片中的金色烟花

### 2. 柳树型 (Willow)
- 向下垂落效果
- 粒子生命周期长
- 重力效果明显
- 类似图片中右上角的粉色烟花

### 3. 棕榈型 (Palm)
- 向上散开后下落
- 形成棕榈树状
- 类似图片中左下角的粉色烟花

### 4. 环形 (Ring)
- 水平扩散
- 形成圆环状
- 粒子密集
- 类似图片中间的橙色烟花

### 5. 菊花型 (Chrysanthemum)
- 密集绽放
- 粒子数量多
- 360度均匀分布
- 类似图片左上角的金色烟花

## 实现细节

### 1. 发射阶段
```swift
private func launchFirework(at targetPosition: CGPoint) {
    // 从屏幕底部随机位置发射
    let startX = CGFloat.random(in: view.bounds.width * 0.2...view.bounds.width * 0.8)
    let startPosition = CGPoint(x: startX, y: view.bounds.height + 20)
    
    // 创建上升的火箭粒子
    let rocketLayer = CAEmitterLayer()
    rocketLayer.emitterPosition = startPosition
    
    // 动画上升到目标位置
    let animation = CABasicAnimation(keyPath: "emitterPosition")
    animation.fromValue = startPosition
    animation.toValue = targetPosition
    animation.duration = duration
    animation.timingFunction = CAMediaTimingFunction(name: .easeOut)
    
    // 到达顶点后爆炸
    DispatchQueue.main.asyncAfter(deadline: .now() + duration) {
        rocketLayer.removeFromSuperlayer()
        self.createFireworkExplosion(at: targetPosition)
    }
}
```

### 2. 爆炸阶段
```swift
private func createFireworkExplosion(at position: CGPoint) {
    // 随机选择烟花类型
    let types: [FireworkType] = [.burst, .willow, .palm, .ring, .chrysanthemum]
    let type = types.randomElement() ?? .burst
    
    createFirework(at: position, type: type)
}
```

## 首页卡片循环动画

### 实现方式
```swift
private func startFireworksAnimation() {
    // 创建烟花绽放粒子效果
    let emitter = CAEmitterLayer()
    emitter.emitterPosition = CGPoint(
        x: containerView.bounds.width / 2,
        y: containerView.bounds.height / 2  // 中心绽放
    )
    
    // 360度全方向爆发
    cell.emissionRange = .pi * 2
    cell.velocity = 60
    cell.birthRate = 50
    
    // 2秒后重新开始，形成循环
    DispatchQueue.main.asyncAfter(deadline: .now() + 2.0) { [weak self] in
        self?.startFireworksAnimation()
    }
}
```

### 循环特点
- 从卡片中心向外绽放
- 360度全方向扩散
- 2秒一个循环
- 自动重复播放
- 粒子快速消失后重新绽放
