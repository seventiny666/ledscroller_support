# GlowLed 演示指南

## 🎬 完整演示流程（5 分钟）

### 准备工作
1. 打开项目：`open GlowLed.xcodeproj`
2. 选择模拟器：iPhone 14 Pro
3. 点击 Run (⌘R)
4. 等待应用启动

---

## 📱 演示场景

### 场景 1：表白模式（1 分钟）

**目标**：展示赛博朋克风格的表白效果

1. **加载预设**
   - 点击"表白 - I LOVE U"按钮
   - 观察参数自动填充

2. **预览效果**
   - 点击"🚀 全屏预览"
   - 看到粉色霓虹文字 "I LOVE U"
   - 粉色爱心从底部上升
   - 点击屏幕退出

3. **生成代码**
   - 点击"💻 生成代码"
   - 选择"查看代码"
   - 展示生成的伪代码：
   ```swift
   GlowLed.create {
       text("I LOVE U")
       .fontSize(72)
       .neonColor("#FF1493")
       .glow(0.9)
       .background("#000000")
       
       effects {
           cyber_heart(density: 0.6)
       }
   }
   ```

**演示要点**：
- ✨ 霓虹发光效果
- 💖 爱心特效
- 💻 代码生成功能

### 场景 4：自定义创作（2 分钟）

**目标**：展示完整的自定义流程

1. **输入文字**
   - 在文本框输入："CYBER PUNK 2077"

2. **调整样式**
   - 拖动"字体大小"到 60
   - 点击蓝色颜色按钮 (#00FFFF)
   - 拖动"霓虹强度"到 0.9

3. **选择动画**
   - 点击"上滚"分段控件
   - 拖动"动画速度"到 2.0

4. **添加特效**
   - 点击"☄️ 流星"按钮（按钮变色表示已启用）
   - 点击"🎆 烟花"按钮（同时启用两个特效）

5. **预览效果**
   - 点击"🚀 全屏预览"
   - 观察：
     - 蓝色霓虹文字从下往上滚动
     - 流星从上往下划过
     - 烟花随机爆炸
   - 点击屏幕退出

6. **生成代码**
   - 点击"💻 生成代码"
   - 选择"复制伪代码"
   - 提示"伪代码已复制"
   - 打开备忘录粘贴查看

**演示要点**：
- 🎨 完整的自定义流程
- 🎭 多特效组合
- 📋 代码复制分享

## 🎨 视觉效果说明

### 霓虹发光效果
- **低强度 (0.3)**：淡淡的光晕
- **中强度 (0.6)**：明显的发光
- **高强度 (0.9)**：强烈的霓虹效果

### 特效密度
- **低密度 (0.3)**：稀疏的粒子
- **中密度 (0.6)**：适中的粒子数量
- **高密度 (0.9)**：密集的粒子（可能影响性能）

## 🐛 演示中可能遇到的问题

### 问题 1：特效看不清
**原因**：背景色和特效颜色对比度不够  
**解决**：使用黑色背景 (#000000)

### 问题 2：动画卡顿
**原因**：特效密度过高或模拟器性能不足  
**解决**：降低特效密度或使用真机演示

### 问题 3：文字太小看不清
**原因**：字体大小设置过小  
**解决**：调整到 60-80 之间

### 问题 4：代码生成后看不到
**原因**：需要点击"查看代码"  
**解决**：在弹出菜单中选择"查看代码"

## 🎓 演示后的 Q&A

### 常见问题

**Q: 支持自定义字体吗？**  
A: 当前版本使用系统字体，未来版本会支持自定义字体。

**Q: 可以保存配置吗？**  
A: 可以通过代码生成功能保存，复制代码到备忘录即可。

**Q: 支持导入代码吗？**  
A: 当前版本暂不支持，但已经实现了解析逻辑，未来版本会添加。

**Q: 特效可以自定义吗？**  
A: 当前版本提供 4 种预设特效，密度可调。未来会添加更多特效。

**Q: 支持 iPad 吗？**  
A: 支持，但界面未针对 iPad 优化。

**Q: 需要联网吗？**  
A: 不需要，完全离线使用。

**Q: 会消耗很多电量吗？**  
A: 全屏预览时会保持屏幕常亮，建议在充电时使用。
