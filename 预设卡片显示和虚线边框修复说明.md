# 预设卡片显示和虚线边框修复说明

## 问题描述

### 问题1：预设卡片缺失
首页"其他分类"中缺少"I ❤️ U"和"I LOVE U"两个预设卡片。

### 问题2：虚线边框不显示
创作页面的"创建LED"按钮虚线边框在初始加载时不显示。

## 问题原因

### 问题1原因
旧的数据合并逻辑：
```swift
// ❌ 旧逻辑：只添加缺失的卡片
for defaultItem in defaultItems {
    if !mergedItems.contains(where: { $0.id == defaultItem.id }) {
        mergedItems.append(defaultItem)
    }
}
```

如果旧数据中已经存在这些ID的卡片（但没有`isDefaultPreset`标记），就不会被更新，导致这些卡片不会显示在"其他分类"中。

### 问题2原因
虚线边框在`layoutSubviews`中创建，但初始加载时`containerView.bounds`可能还是零，导致虚线路径无效。

## 修复方案

### 1. 修复数据合并逻辑

**文件**: `GlowLed/LEDItem.swift`

```swift
func loadItems() -> [LEDItem] {
    guard let data = userDefaults.data(forKey: key),
          let savedItems = try? JSONDecoder().decode([LEDItem].self, from: data) else {
        return getDefaultItems()
    }
    
    let defaultItems = getDefaultItems()
    var mergedItems: [LEDItem] = []
    
    // ✅ 首先添加所有默认卡片（包括更新的预设卡片）
    for defaultItem in defaultItems {
        mergedItems.append(defaultItem)
    }
    
    // ✅ 然后添加用户创建的卡片（排除默认卡片的ID）
    let defaultIds = Set(defaultItems.map { $0.id })
    for savedItem in savedItems {
        if !defaultIds.contains(savedItem.id) {
            mergedItems.append(savedItem)
        }
    }
    
    return mergedItems
}
```

**修复逻辑**：
1. 先添加所有默认卡片（确保最新的定义）
2. 再添加用户创建的卡片（排除默认卡片ID）
3. 这样可以确保默认卡片始终使用最新的定义（包括`isDefaultPreset`标记）

### 2. 修复虚线边框显示

**文件**: `GlowLed/MyCreationsViewController.swift`

```swift
class AddCreationCell: UICollectionViewCell {
    private var dashedBorder: CAShapeLayer?
    
    override func layoutSubviews() {
        super.layoutSubviews()
        updateDashedBorder()
    }
    
    private func updateDashedBorder() {
        // 移除旧的虚线边框
        dashedBorder?.removeFromSuperlayer()
        
        // ✅ 确保containerView有有效的bounds
        guard containerView.bounds.width > 0 && containerView.bounds.height > 0 else {
            return
        }
        
        // 创建新的虚线边框
        let shapeLayer = CAShapeLayer()
        shapeLayer.strokeColor = UIColor.systemGray3.cgColor
        shapeLayer.lineWidth = 2
        shapeLayer.lineDashPattern = [6, 4]
        shapeLayer.fillColor = nil
        shapeLayer.path = UIBezierPath(roundedRect: containerView.bounds, cornerRadius: 16).cgPath
        containerView.layer.insertSublayer(shapeLayer, at: 0) // ✅ 插入到最底层
        
        dashedBorder = shapeLayer
    }
}
```

**修复要点**：
1. 提取`updateDashedBorder()`方法
2. 检查`bounds`是否有效（宽高大于0）
3. 使用`insertSublayer(at: 0)`插入到最底层，避免遮挡内容
