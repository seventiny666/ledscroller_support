# 图片资源说明

## 图片存放位置
所有模版图片应放在 Xcode 项目的 `Assets.xcassets` 文件夹中。

## 图片命名规则

### 霓虹灯看板模版（4张）
- `neon_1.png` - 霓虹灯看板模版1
- `neon_2.png` - 霓虹灯看板模版2
- `neon_3.png` - 霓虹灯看板模版3
- `neon_4.png` - 霓虹灯看板模版4

### 偶像应援模版（4张）
- `idol_1.png` - 偶像应援模版1
- `idol_2.png` - 偶像应援模版2
- `idol_3.png` - 偶像应援模版3
- `idol_4.png` - 偶像应援模版4

### LED屏幕模版（4张）
- `led_1.png` - LED屏幕模版1
- `led_2.png` - LED屏幕模版2
- `led_3.png` - LED屏幕模版3
- `led_4.png` - LED屏幕模版4

## 图片规格要求

### 尺寸比例
- 宽高比：16:9
- 推荐尺寸：1920x1080 或 1280x720
- 最小尺寸：640x360

### 文件格式
- 格式：PNG（推荐）或 JPG
- 颜色模式：RGB
- 透明度：支持（PNG格式）

### 文件大小
- 单张图片建议不超过 500KB
- 使用压缩工具优化图片大小

## 如何添加图片到Xcode

### 方法1：通过Xcode添加
1. 打开 Xcode 项目
2. 在左侧导航器中找到 `Assets.xcassets`
3. 右键点击 `Assets.xcassets`
4. 选择 "New Image Set"
5. 将图片拖拽到对应的位置（1x, 2x, 3x）
6. 重命名 Image Set 为对应的名称（如 `neon_1`）

### 方法2：直接拖拽
1. 打开 Xcode 项目
2. 在左侧导航器中点击 `Assets.xcassets`
3. 直接将图片文件拖拽到 Assets 列表中
4. 确保图片名称符合命名规则

## 占位符说明

在没有实际图片的情况下，应用会显示：
- 霓虹灯看板：深蓝色背景 (#1a1a2e)
- 偶像应援：深蓝色背景 (#1a1a2e)
- LED屏幕：深蓝色背景 (#1a1a2e)

## 图片内容建议

### 霓虹灯看板
- 霓虹灯效果的文字或图案
- 发光效果明显
- 背景较暗，突出霓虹效果
- 可以包含：店铺招牌、艺术字、装饰图案

### 偶像应援
- 应援文字（如：加油、冲鸭）
- 偶像名字或口号
- 明亮的颜色（粉色、蓝色、紫色等）
- 可以包含：应援棒效果、星星装饰

### LED屏幕
- 电子屏幕效果
- 滚动字幕样式
- 点阵效果
- 可以包含：通知信息、广告文字、倒计时

## 注意事项

1. 所有图片必须符合版权要求
2. 图片内容应积极向上，不包含敏感内容
3. 图片清晰度要高，避免模糊
4. 建议使用统一的视觉风格
5. 图片文件名不要包含中文或特殊字符
6. 确保图片在不同设备上显示正常
