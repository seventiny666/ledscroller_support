```swift

    startAnimation()

    super.viewWillDisappear(animated)
    
    // 停止动画和屏幕常亮
    UIApplication.shared.isIdleTimerDisabled = false
    textLabel.layer.removeAllAnimations()
    
    // 恢复为竖屏
    AppDelegate.orientationLock = .portrait
}
```

## 调试步骤

### 步骤1：运行应用并查看控制台

1. 在 Xcode 中运行应用
2. 打开控制台（⌘⇧Y）
3. 进入首页

**预期输出：**
```
📱 首页 viewWillAppear
   collectionView.isUserInteractionEnabled: true
   view.isUserInteractionEnabled: true
```

### 步骤2：测试创建按钮

1. 点击右上角"+"按钮

**预期输出：**
```
🔍 点击了创建按钮
```

**如果没有输出：**
- 导航栏按钮被禁用
- 检查 navigationItem.rightBarButtonItem

### 步骤3：测试卡片点击

1. 点击任意LED卡片

**预期输出：**
```
🔍 点击了卡片: 0
```

**如果没有输出：**
- collectionView 交互被禁用
- 卡片被其他视图遮挡
- 触摸事件被拦截

### 步骤4：测试长按

1. 长按任意LED卡片
2. 应该弹出菜单

**如果没有弹出：**
- contextMenu 配置有问题
- 长按手势被禁用

## 可能的问题和解决方案

### 问题1：控制台显示 isUserInteractionEnabled: false

**原因：** 视图交互被禁用

**解决方案：**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)
    
    // 确保交互启用
    view.isUserInteractionEnabled = true
    collectionView.isUserInteractionEnabled = true
    
    AppDelegate.orientationLock = .portrait
    loadData()
}
```

---

### 问题2：点击卡片没有任何反应

**原因：** collectionView 被遮挡或触摸事件被拦截

**解决方案：**
```swift
override func viewDidAppear(_ animated: Bool) {
    super.viewDidAppear(animated)
    
    // 调试视图层级
    print("View hierarchy:")
    print("  view: \(view)")
    print("  collectionView: \(collectionView)")
    print("  collectionView.superview: \(collectionView.superview)")
    
    // 确保 collectionView 在最上层
    view.bringSubviewToFront(collectionView)
}
```

---

### 问题3：创建按钮无法点击

**原因：** 导航栏按钮被禁用

**解决方案：**
```swift
override func viewWillAppear(_ animated: Bool) {
    super.viewWillAppear(animated)
    
    // 重新设置导航栏按钮
    let createButton = UIBarButtonItem(barButtonSystemItem: .add, target: self, action: #selector(createLED))
    createButton.tintColor = .systemPink
    createButton.isEnabled = true
    navigationItem.rightBarButtonItem = createButton
    
    AppDelegate.orientationLock = .portrait
    loadData()
}
```

---

### 问题4：屏幕方向不对

**原因：** AppDelegate.orientationLock 没有生效

**检查 AppDelegate：**
```swift
func application(_ application: UIApplication, 
                supportedInterfaceOrientationsFor window: UIWindow?) 
                -> UIInterfaceOrientationMask {
    print("🔄 orientationLock: \(orientationLock)")
    return orientationLock
}
```

**检查 Info.plist：**
```xml
<key>UISupportedInterfaceOrientations</key>
<array>
    <string>UIInterfaceOrientationPortrait</string>
    <string>UIInterfaceOrientationLandscapeLeft</string>
    <string>UIInterfaceOrientationLandscapeRight</string>
</array>
```

---

### 问题5：第一次可以点击，第二次不行

**原因：** 视图状态没有正确恢复

**解决方案：**
```swift
override func viewDidAppear(_ animated: Bool) {
    super.viewDidAppear(animated)
    
    // 强制刷新视图
    view.setNeedsLayout()
    view.layoutIfNeeded()
    
    // 重新加载数据
    collectionView.reloadData()
}
```
