# 滚动动画预览功能说明

## 功能概述

在创建LED页面的预览区域实现滚动动画效果，让用户能够实时看到滚动方式和速度的效果。

## 实现方式

使用CABasicAnimation实现四个方向的滚动动画：
- 左滚：从右侧进入，向左滚动到左侧消失
- 右滚：从左侧进入，向右滚动到右侧消失
- 上滚：从下方进入，向上滚动到上方消失
- 下滚：从上方进入，向下滚动到下方消失

## 技术实现

### 1. 核心方法

```swift
private func applyScrollAnimation() {
    // 移除旧的动画
    previewLabel.layer.removeAllAnimations()
    
    // 获取滚动类型
    let scrollTypes: [LEDItem.ScrollType] = [.none, .scrollLeft, .scrollRight, .scrollUp, .scrollDown]
    let scrollType = scrollTypes[scrollTypeSegment.selectedSegmentIndex]
    
    // 如果是静止，不添加动画
    if scrollType == .none {
        previewLabel.transform = .identity
        return
    }
    
    // 计算动画时长（速度越快，时长越短）
    let baseDuration: TimeInterval = 3.0
    let duration = baseDuration / Double(currentItem.speed)
    
    // 根据滚动类型创建动画
    switch scrollType {
    case .scrollLeft:
        animateScrollLeft(duration: duration)
    case .scrollRight:
        animateScrollRight(duration: duration)
    case .scrollUp:
        animateScrollUp(duration: duration)
    case .scrollDown:
        animateScrollDown(duration: duration)
    default:
        break
    }
}
```

### 2. 左滚动画

```swift
private func animateScrollLeft(duration: TimeInterval) {
    let containerWidth = previewContainer.bounds.width
    let labelWidth = previewLabel.intrinsicContentSize.width
    
    // 从右侧进入，向左滚动到左侧消失
    let startX = containerWidth / 2 + labelWidth / 2
    let endX = -labelWidth / 2
    
    previewLabel.layer.removeAllAnimations()
    
    let animation = CABasicAnimation(keyPath: "position.x")
    animation.fromValue = startX
    animation.toValue = endX
    animation.duration = duration
    animation.repeatCount = .infinity
    animation.timingFunction = CAMediaTimingFunction(name: .linear)
    
    previewLabel.layer.add(animation, forKey: "scrollLeft")
}
```

## 触发时机

滚动动画在以下情况下会被触发：

### 1. 切换滚动方式
```swift
@objc private func scrollTypeChanged() {
    updateSpeedVisibility()
    updateCurrentItem()
    updatePreview()
    applyScrollAnimation() // ✅ 触发动画
}
```

### 2. 调整播放速度
```swift
@objc private func speedChanged() {
    currentItem.speed = CGFloat(speedSlider.value)
    speedLabel.text = "播放速度: \(String(format: "%.1f", currentItem.speed))"
    applyScrollAnimation() // ✅ 触发动画
}
```

### 3. 更新预览
```swift
private func updatePreview() {
    // ... 更新文字、颜色、背景等
    
    // 应用滚动动画
    DispatchQueue.main.asyncAfter(deadline: .now() + 0.1) {
        self.applyScrollAnimation() // ✅ 触发动画
    }
}
```

### 4. 加载数据
```swift
private func loadCurrentData() {
    // ... 加载数据
    
    // 应用滚动动画
    DispatchQueue.main.asyncAfter(deadline: .now() + 0.2) {
        self.applyScrollAnimation() // ✅ 触发动画
    }
}
```

## 动画参数

### 基础时长
```swift
let baseDuration: TimeInterval = 3.0
```
- 基础时长为3秒
- 速度为1.0时，完整滚动一次需要3秒

### 实际时长
```swift
let duration = baseDuration / Double(currentItem.speed)
```
- 速度0.5：6秒（慢速）
- 速度1.0：3秒（正常）
- 速度1.5：2秒（快速）
- 速度2.0：1.5秒（很快）
- 速度3.0：1秒（极快）

### 动画属性
- **repeatCount**: `.infinity`（无限循环）
- **timingFunction**: `.linear`（线性，匀速）
- **keyPath**: `position.x` 或 `position.y`（位置动画）

## 动画计算

### 水平滚动（左/右）

**左滚**:
```
起始位置 = 容器宽度/2 + 文字宽度/2  (右侧外)
结束位置 = -文字宽度/2              (左侧外)
```

**右滚**:
```
起始位置 = -文字宽度/2              (左侧外)
结束位置 = 容器宽度/2 + 文字宽度/2  (右侧外)
```

### 垂直滚动（上/下）

**上滚**:
```
起始位置 = 容器高度/2 + 文字高度/2  (下方外)
结束位置 = -文字高度/2              (上方外)
```

**下滚**:
```
起始位置 = -文字高度/2              (上方外)
结束位置 = 容器高度/2 + 文字高度/2  (下方外)
```

## 性能优化

### 1. 移除旧动画
```swift
previewLabel.layer.removeAllAnimations()
```
- 在添加新动画前，先移除旧动画
- 避免动画叠加导致的性能问题

### 2. 使用CABasicAnimation
- 硬件加速
- 性能优秀
- 流畅度高

### 3. 延迟执行
```swift
DispatchQueue.main.asyncAfter(deadline: .now() + 0.1) {
    self.applyScrollAnimation()
}
```
- 等待布局完成后再应用动画
- 确保尺寸计算准确

## 注意事项

### 1. 布局时机
- 动画需要在布局完成后应用
- 使用`asyncAfter`延迟执行
- 确保容器尺寸已确定

### 2. 动画清理
- 切换滚动方式时清除旧动画
- 静止模式时移除所有动画
- 避免内存泄漏

### 3. 文字尺寸
- 使用`intrinsicContentSize`获取文字尺寸
- 动态计算起始和结束位置
- 适应不同长度的文字
