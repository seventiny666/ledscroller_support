# 启动动画优化说明

## 修改内容

简化了首次启动动画，使用应用的AppIcon作为启动Logo，缩短启动时长，提升用户体验。

## 修改文件

- `LedScreen/SceneDelegate.swift` - LaunchAnimationViewController类
- `LedScreen/Assets.xcassets/LaunchLogo.imageset/` - 新增启动Logo资源

## 优化前后对比

### 之前的启动动画
- 复杂的多阶段动画
- 弹性放大 + 连续脉动 + 颜色变化
- 总时长：约3.5秒
- 视觉效果：炫酷但耗时较长

### 优化后的启动动画
- 简洁的淡入淡出动画
- 上方：圆角Logo图片（应用的AppIcon，120x120）
- 下方：应用名称"LedScreen"（主题色）
- 总时长：约1.2秒
- 视觉效果：简洁专业

## 新动画流程

```
0.0s - 黑色背景
0.1s - 开始淡入（0.5秒）
0.6s - 完全显示
0.9s - 开始淡出（0.4秒）
1.3s - 完成，进入主界面
```

## Logo资源

### LaunchLogo.imageset
- 位置：`LedScreen/Assets.xcassets/LaunchLogo.imageset/`
- 来源：从AppIcon复制而来
- 包含：
  - LaunchLogo@2x.png (120x120)
  - LaunchLogo@3x.png (180x180)
- 优势：可以直接通过`UIImage(named: "LaunchLogo")`访问

### 为什么不直接使用AppIcon？
AppIcon.appiconset中的图片在iOS中无法直接通过`UIImage(named:)`访问，需要创建独立的imageset。

## 样式参数

### Logo
- 尺寸：120x120
- 圆角：30px
- 位置：居中偏上50px
- 来源：应用的AppIcon

### 文字
- 内容："LedScreen"
- 字体：System Semibold 32pt
- 颜色：主题色 (#8EFFE6)
- 位置：Logo下方24px

### 动画
- 淡入：0.5秒（EaseOut）
- 停留：0.3秒
- 淡出：0.4秒（EaseIn）
- 总时长：1.2秒

## 性能优化

1. **时长缩短**：从3.5秒减少到1.2秒（减少66%）
2. **动画简化**：去掉复杂的弹性、脉动、颜色变化
3. **资源优化**：使用系统动画，无额外图层
4. **用户体验**：快速进入应用，减少等待

## 首次启动标记

应用使用UserDefaults记录首次启动：
```swift
UserDefaults.standard.bool(forKey: "HasLaunchedBefore")
```

- 首次启动：显示启动动画
- 后续启动：直接进入主界面

## 测试建议

### 重置首次启动状态
在模拟器或真机上测试启动动画：

1. **方法1：删除应用重装**
   - 删除应用
   - 重新运行

2. **方法2：清除UserDefaults**
   ```swift
   UserDefaults.standard.removeObject(forKey: "HasLaunchedBefore")
   ```

3. **方法3：重置模拟器**
   - Device → Erase All Content and Settings

## 更新AppIcon

如果需要更新启动Logo：

1. 替换AppIcon.appiconset中的图标文件
2. 重新复制到LaunchLogo.imageset：
   ```bash
   cp "LedScreen/Assets.xcassets/AppIcon.appiconset/icon_60pt@2x.png" \
      "LedScreen/Assets.xcassets/LaunchLogo.imageset/LaunchLogo@2x.png"
   
   cp "LedScreen/Assets.xcassets/AppIcon.appiconset/icon_60pt@3x.png" \
      "LedScreen/Assets.xcassets/LaunchLogo.imageset/LaunchLogo@3x.png"
   ```
3. 重新编译运行
