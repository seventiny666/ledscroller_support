# 边框保存问题修复总结

## 问题描述
成为订阅用户后，从首页选择带有VIP标签的模板（如"Marry Me"），通过编辑按钮进入LED屏幕编辑页面，保存后在"我的创作"模块中，卡片的封面和全屏预览都没有显示首页的边框效果。

## 根本原因
在 `LEDCreateViewController.swift` 的 `saveTapped()` 方法中，当处于模板编辑模式（`isTemplateEdit = true`）时，创建新的 `LEDItem` 对象时遗漏了 `linearBorderStyle` 参数，导致线性边框信息丢失。

## 修复内容

### 1. LEDCreateViewController.swift
**修复位置**: `saveTapped()` 方法中的模板编辑模式分支

**修改前**:
```swift
let newItem = LEDItem(
    id: UUID().uuidString,
    text: currentItem.text,
    fontSize: currentItem.fontSize,
    textColor: currentItem.textColor,
    backgroundColor: currentItem.backgroundColor,
    backgroundImageName: currentItem.backgroundImageName,
    glowIntensity: currentItem.glowIntensity,
    scrollType: currentItem.scrollType,
    speed: currentItem.speed,
    fontName: currentItem.fontName,
    borderStyle: currentItem.borderStyle, // 保存跑马灯边框
    lightBoardStyle: currentItem.lightBoardStyle // 保存灯牌边框
)
```

**修改后**:
```swift
let newItem = LEDItem(
    id: UUID().uuidString,
    text: currentItem.text,
    fontSize: currentItem.fontSize,
    textColor: currentItem.textColor,
    backgroundColor: currentItem.backgroundColor,
    backgroundImageName: currentItem.backgroundImageName,
    glowIntensity: currentItem.glowIntensity,
    scrollType: currentItem.scrollType,
    speed: currentItem.speed,
    fontName: currentItem.fontName,
    borderStyle: currentItem.borderStyle, // 保存跑马灯边框
    lightBoardStyle: currentItem.lightBoardStyle, // 保存灯牌边框
    linearBorderStyle: currentItem.linearBorderStyle // 保存线性边框
)
```

### 2. LEDPreviewViewController.swift
**问题**: 预览页面缺少线性边框支持

**修复内容**:
- 添加 `linearBorderView` 属性
- 在 `setupUI()` 中添加线性边框视图的初始化和约束
- 在边框设置逻辑中添加线性边框的处理分支

**关键代码**:
```swift
// 添加属性
private let linearBorderView = LinearBorderView(displayMode: .preview)

// 添加视图和约束
linearBorderView.translatesAutoresizingMaskIntoConstraints = false
linearBorderView.isHidden = true
previewContainer.addSubview(linearBorderView)

// 添加边框处理逻辑
} else if let linearBorderStyleIndex = ledItem.linearBorderStyle,
          linearBorderStyleIndex >= 0 && linearBorderStyleIndex < LinearBorderStyle.allCases.count {
    let style = LinearBorderStyle.allCases[linearBorderStyleIndex]
    linearBorderView.setStyle(style)
    linearBorderView.isHidden = false
    borderView.isHidden = true
    lightBoardView.isHidden = true
}
```

- ✅ LEDPreviewViewController: 预览页面支持线性边框显示
- ✅ LEDFullScreenViewController: 全屏页面已支持线性边框（无需修改）

## 影响范围
- 从首页模板编辑并保存的所有LED项目
- 三种边框类型：跑马灯边框、灯牌边框、线性边框
