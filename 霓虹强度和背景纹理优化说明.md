# 霓虹强度和背景纹理优化说明

## 修复时间
2026-03-01

## 优化内容

### 1. 霓虹强度范围扩展 ✅
- 从原来的 0-1 扩展到 0-5
- 提供更强的发光效果选择
- 预览区和全屏显示都已适配新范围

**计算公式：**
- 预览区：`glowRadius = 5 * glowIntensity` (0-25范围)
- 全屏显示：`glowRadius = 10 * glowIntensity` (0-50范围)
- 透明度：`glowOpacity = min(glowIntensity / 5.0, 1.0)` (归一化到0-1)

**默认值更新：**
- 新建LED默认霓虹强度：2.5
- 预设LED霓虹强度：2.8 - 5.0 之间

### 2. 背景选择增加纹理行 ✅
在纯色背景下方增加了一行纹理背景选择：

**纹理类型：**
1. **stars** - 深蓝星空 (#0C0C26)
2. **gradient** - 紫色渐变 (#33194D)
3. **dots** - 深蓝点阵 (#192633)
4. **waves** - 青色波浪 (#00334D)
5. **grid** - 灰色网格 (#262626)
6. **noise** - 噪点背景 (#1F1F2E)

**布局：**
- 第一行：6个纯色背景 (4:3比例，48x36)
- 第二行：6个纹理背景 (4:3比例，48x36)
- 间距：8px
- 选择时会清除另一行的选择状态

### 3. 文字颜色按钮优化 ✅
- 确认为圆形设计 (36x36，圆角18)
- 使用 `fillEqually` 分布确保均匀排列
- 间距：12px

### 4. 界面滚动优化 ✅
界面布局符合用户常规操作习惯：

**内容顺序：**
1. 预览区 (120px高度)
2. 文字内容输入框
3. 字体大小滑块
4. 字体选择 (苹方/黑体/宋体/楷体)
5. 文字颜色选择 (圆形按钮)
6. 霓虹强度滑块 (0-5)
7. 选择背景 (两行：纯色+纹理)
8. 滚动方式选择
9. 滚动速度滑块

**滚动体验：**
- 使用 UIScrollView 支持流畅滚动
- 键盘弹出时自动调整内容偏移
- 所有控件都在可触达范围内

## 技术实现

### UIColor扩展增强
添加了 `toHex()` 方法，支持将UIColor转换回十六进制字符串：

```swift
func toHex() -> String? {
    guard let components = cgColor.components, components.count >= 3 else {
        return nil
    }
    
    let r = Float(components[0])
    let g = Float(components[1])
    let b = Float(components[2])
    
    return String(format: "#%02lX%02lX%02lX",
                 lroundf(r * 255),
                 lroundf(g * 255),
                 lroundf(b * 255))
}
```

### 纹理背景创建
```swift
private func createTextureBackground(type: String) -> UIColor {
    switch type {
    case "stars":
        return UIColor(red: 0.05, green: 0.05, blue: 0.15, alpha: 1)
    case "gradient":
        return UIColor(red: 0.2, green: 0.1, blue: 0.3, alpha: 1)
    // ... 其他纹理
    default:
        return .black
    }
}
```

### 纹理按钮处理
```swift
@objc private func textureButtonTapped(_ sender: UIButton) {
    let textureIndex = sender.tag - 300
    let textureName = bgTextures[textureIndex]
    
    // 将纹理颜色转换为hex存储
    let textureColor = createTextureBackground(type: textureName)
    currentItem.backgroundColor = textureColor.toHex() ?? "#000000"
    
    // 更新选择状态
    updateColorButtonSelection(tag: 300, selectedIndex: textureIndex)
    // 清除纯色选择
    updateColorButtonSelection(tag: 200, selectedIndex: -1)
    
    updatePreview()
}
```
