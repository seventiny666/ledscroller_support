# GlowLed - LED广场

一个赛博朋克风格的iOS LED屏幕应用，支持自定义文字、霓虹发光效果和多种滚动动画。

## ✨ 主要功能

### LED广场
- 网格布局展示所有LED屏幕
- 圆角矩形卡片设计
- 实时预览缩略图
- 点击全屏播放
- 长按编辑或删除

### 创建LED屏幕
- 📝 自定义文字内容
- 🎨 7种霓虹颜色选择
- 📏 可调节字体大小 (30-100)
- ✨ 可调节发光强度
- 🎭 6种背景颜色
- 🔄 5种滚动方式 (静止/左滚/右滚/上滚/下滚)
- ⚡ 可调节滚动速度
- 👁️ 实时预览功能

### 全屏播放
- 全屏沉浸式体验
- 流畅的滚动动画
- 屏幕常亮
- 点击退出

### 数据管理
- 自动保存所有创建的LED屏幕
- 支持编辑已保存的屏幕
- 支持删除不需要的屏幕
- 数据持久化存储

## 📱 系统要求

- iOS 15.0+
- Xcode 15.0+
- Swift 5.9+

## 🚀 安装运行

1. 克隆或下载项目
2. 在 Xcode 中打开 `GlowLed.xcodeproj`
3. 选择模拟器或真机
4. 点击运行按钮

## 📖 使用说明

1. **启动应用** - 进入LED广场首页
2. **创建LED屏幕** - 点击右上角"+"按钮
3. **自定义设置**:
   - 输入想要显示的文字
   - 选择文字颜色和字体大小
   - 调节霓虹发光强度
   - 选择背景颜色
   - 选择滚动方式和速度
4. **预览效果** - 点击"预览效果"按钮查看
5. **保存** - 点击右上角"保存"按钮
6. **播放** - 在首页点击LED屏幕全屏播放
7. **编辑/删除** - 长按LED屏幕选择操作
