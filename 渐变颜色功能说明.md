# 渐变颜色功能说明

## 功能概述

在创建LED页面中，为文字颜色选择添加了渐变颜色功能，提供更丰富的视觉效果。

## 新增功能

### 1. 文字颜色扩展
- **原来**: 1行7个纯色
- **现在**: 2行14个纯色
  - 第一行: #FF00FF, #00FFFF, #FFD700, #FF1493, #00FF00, #FF4500, #FFFFFF
  - 第二行: #FF69B4, #8B00FF, #00CED1, #FFB6C1, #32CD32, #FF8C00, #F0E68C

### 2. 渐变颜色（新增）
在文字颜色下方新增2行渐变颜色选择：

#### 第一行 - 经典渐变
1. 粉红→橙色 (#FF0080 → #FF8C00)
2. 青色→蓝色 (#00F5FF → #0080FF)
3. 金色→粉色 (#FFD700 → #FF1493)
4. 绿色→青色 (#00FF00 → #00CED1)
5. 紫红→紫色 (#FF00FF → #8B00FF)
6. 橙红→金色 (#FF4500 → #FFD700)
7. 粉色→青色 (#FF1493 → #00FFFF)

#### 第二行 - 霓虹渐变
1. 玫红→紫色 (#FF006E → #8338EC)
2. 绿色→黄色 (#06FFA5 → #FFFB00)
3. 粉红→紫色 (#F72585 → #7209B7)
4. 天蓝→蓝色 (#4CC9F0 → #4361EE)
5. 玫红→黄色 (#FF006E → #FFBE0B)
6. 绿色→蓝色 (#06FFA5 → #3A86FF)
7. 粉红→天蓝 (#F72585 → #4CC9F0)

### 2. 渐变按钮创建
```swift
private func createGradientStack(gradients: [[String]], tag: Int) -> UIStackView {
    let buttons = gradients.enumerated().map { index, colors -> UIButton in
        let btn = UIButton(type: .system)
        
        // 创建渐变层
        let gradientLayer = CAGradientLayer()
        gradientLayer.colors = colors.map { UIColor(hex: $0).cgColor }
        gradientLayer.startPoint = CGPoint(x: 0, y: 0.5)
        gradientLayer.endPoint = CGPoint(x: 1, y: 0.5)
        gradientLayer.frame = CGRect(x: 0, y: 0, width: 36, height: 36)
        gradientLayer.cornerRadius = 18
        
        btn.layer.insertSublayer(gradientLayer, at: 0)
        // ... 其他配置
        
        return btn
    }
    // ...
}
```

### 4. 选择逻辑
- 选择纯色时：清除渐变选择，显示纯色文字
- 选择渐变时：清除纯色选择，应用渐变效果
- 互斥选择：纯色和渐变不能同时选中

## 使用说明

### 选择纯色
1. 点击"文字颜色"下的任意颜色按钮
2. 预览区域显示纯色文字
3. 渐变选择自动清除

### 选择渐变
1. 点击"渐变颜色"下的任意渐变按钮
2. 预览区域显示渐变文字
3. 纯色选择自动清除

### 预览效果
- 实时预览：点击颜色/渐变后立即在预览区域显示
- 发光效果：渐变文字也支持霓虹发光效果
- 字体大小：渐变随字体大小自动调整

## 按钮标识

### Tag分配
- 100-106: 文字颜色第一行
- 110-116: 文字颜色第二行
- 200-205: 背景颜色
- 300-305: 背景纹理
- 400-406: 渐变颜色第一行
- 410-416: 渐变颜色第二行

### 选中状态
- 选中：边框宽度3px，白色边框
- 未选中：边框宽度2px，30%透明白色边框

## 渐变颜色设计原则

### 1. 适合LED屏幕
- 高对比度：确保在暗背景下清晰可见
- 鲜艳色彩：使用饱和度高的颜色
- 平滑过渡：渐变自然，无明显断层

### 2. 视觉效果
- 经典渐变：常见的配色方案，适合大多数场景
- 霓虹渐变：更鲜艳的配色，适合夜店、派对等场景

### 3. 颜色搭配
- 冷暖对比：如青色→橙色
- 同色系渐变：如粉色→紫色
- 互补色渐变：如绿色→黄色
