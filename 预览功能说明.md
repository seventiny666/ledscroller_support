# 预览功能说明

## 新增功能

在创建/编辑页面的最上方添加了实时预览效果区域。

## 预览区域特性

### 视觉设计
- 位置：页面最顶部，导航栏下方
- 尺寸：宽度占满（左右各留20边距），高度120
- 圆角：15（圆角矩形）
- 背景：实时显示选择的背景颜色

### 预览文字
- 默认显示："预览"
- 输入文字后：实时显示输入的内容
- 字体大小：自动缩放到适合预览容器（最大32）
- 文字颜色：实时显示选择的颜色
- 霓虹效果：实时显示发光强度

## 实时更新功能

### 1. 文字输入 - 实时更新
```swift
输入 "Hello" → 预览立即显示 "Hello"
删除文字 → 预览显示 "预览"
```

### 2. 文字颜色 - 实时更新
```swift
点击颜色按钮 → 预览文字颜色立即改变
```

### 3. 背景颜色 - 实时更新
```swift
点击背景颜色按钮 → 预览背景立即改变
```

### 4. 字体大小 - 实时更新
```swift
滑动字体大小滑块 → 预览文字大小立即改变
（预览中会自动缩放到合适大小）
```

### 5. 霓虹强度 - 实时更新
```swift
滑动霓虹强度滑块 → 预览发光效果立即改变
```

## 技术实现

### 预览容器
```swift
private let previewContainer = UIView()
private let previewLabel = UILabel()
```

### 实时更新方法
```swift
private func updatePreview() {
    // 更新文字
    let displayText = textField.text?.isEmpty == false ? textField.text! : "预览"
    previewLabel.text = displayText
    
    // 更新字体大小（缩小到适合预览）
    let previewFontSize = min(currentItem.fontSize * 0.5, 32)
    previewLabel.font = .boldSystemFont(ofSize: previewFontSize)
    
    // 更新颜色
    previewLabel.textColor = UIColor(hex: currentItem.textColor)
    previewContainer.backgroundColor = UIColor(hex: currentItem.backgroundColor)
    
    // 更新霓虹效果
    previewLabel.layer.shadowColor = UIColor(hex: currentItem.textColor).cgColor
    previewLabel.layer.shadowRadius = 15 * currentItem.glowIntensity
    previewLabel.layer.shadowOpacity = Float(currentItem.glowIntensity)
}
```

### 文字输入实时监听
```swift
func textField(_ textField: UITextField, 
              shouldChangeCharactersIn range: NSRange, 
              replacementString string: String) -> Bool {
    // 计算新的文字
    let currentText = textField.text ?? ""
    guard let stringRange = Range(range, in: currentText) else { return true }
    let updatedText = currentText.replacingCharacters(in: stringRange, with: string)
    
    // 实时更新预览
    DispatchQueue.main.async {
        self.previewLabel.text = updatedText.isEmpty ? "预览" : updatedText
    }
    
    return true
}
```

## 用户体验

### 优势
1. **即时反馈** - 所有修改立即可见
2. **所见即所得** - 预览效果与最终效果一致
3. **减少试错** - 不需要反复点击"预览"按钮
4. **提高效率** - 快速调整到满意的效果

### 使用流程
1. 进入创建页面
2. 看到顶部预览区域显示"预览"
3. 输入文字 → 预览立即更新
4. 选择颜色 → 预览立即更新
5. 调整参数 → 预览立即更新
6. 满意后点击"保存"或"预览"（全屏）

## 预览 vs 全屏预览

### 顶部预览（新增）
- 位置：页面顶部
- 尺寸：120高度的小预览
- 方向：竖屏
- 动画：无滚动动画
- 用途：快速查看效果，实时调整

### 全屏预览（原有）
- 位置：全屏
- 尺寸：整个屏幕
- 方向：横屏
- 动画：完整的滚动动画
- 用途：查看最终效果

## 布局调整

### 原布局
```
导航栏

滚动视图（从顶部开始）
  - 文字内容
  - 字体大小

```

### 新布局
```

↓

  - 预览文字（实时更新）
↓
滚动视图（从预览下方开始）
  - 文字内容
  - 字体大小
  - ...
```

## 注意事项

1. **字体大小自动缩放** - 预览中的字体会自动缩放到合适大小，不会超出容器
2. **文字自动换行** - 长文字会自动换行显示
3. **最小缩放** - 设置了 `minimumScaleFactor = 0.5`，确保文字不会太小
4. **性能优化** - 使用 `DispatchQueue.main.async` 确保UI更新流畅
