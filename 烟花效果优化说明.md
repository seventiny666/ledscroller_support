# 烟花效果优化说明

## 优化时间
2026-03-01

## 优化内容

### 1. 烟花效果改为向上散开 ✅

**原效果：** 360度全方向爆炸
**新效果：** 60度扇形向上散开，更符合真实烟花

**关键参数调整：**
```swift
emissionRange = .pi / 3        // 60度扇形（原：π * 2 = 360度）
emissionLongitude = -.pi / 2   // 向上方向-90度（新增）
velocity = 250                 // 初始速度提高（原：200）
yAcceleration = 150            // 重力加强（原：100）
lifetime = 2.5                 // 生命周期延长（原：2.0）
```

**视觉效果：**
- 粒子从点击位置向上喷射
- 呈扇形散开（60度角）
- 受重力影响逐渐下落
- 形成优美的抛物线轨迹
- 更接近真实烟花效果

### 2. 首页烟花卡片动态预览 ✅

**实现方式：** 使用CAEmitterLayer在卡片上添加实时烟花粒子效果

**预览效果特点：**
- 从卡片底部向上发射小型烟花粒子
- 6种彩色粒子（黄、橙、红、粉、紫、蓝）
- 持续动画，自动循环
- 粒子数量适中，不影响性能
- 文字带金色发光效果

**粒子参数（缩小版）：**
```swift
birthRate = 3              // 每秒3个粒子（全屏版：80）
lifetime = 1.5             // 存活1.5秒（全屏版：2.5）
velocity = 40              // 速度40（全屏版：250）
velocityRange = 15         // 速度范围15（全屏版：80）
emissionRange = .pi / 4    // 45度扇形（全屏版：60度）
scale = 0.3                // 粒子大小0.3（全屏版：0.7）
yAcceleration = 30         // 轻微重力（全屏版：150）
```

## 技术实现

### 向上散开烟花
```swift
private func createFireworkCell(color: UIColor) -> CAEmitterCell {
    let cell = CAEmitterCell()
    cell.contents = createParticleImage(color: color).cgImage
    
    // 向上散开配置
    cell.birthRate = 80
    cell.lifetime = 2.5
    cell.velocity = 250
    cell.velocityRange = 80
    cell.emissionRange = .pi / 3      // 60度扇形
    cell.emissionLongitude = -.pi / 2 // 向上（-90度）
    
    cell.scale = 0.7
    cell.scaleSpeed = -0.25
    cell.alphaSpeed = -0.4
    
    cell.yAcceleration = 150 // 重力
    cell.spin = 3
    cell.spinRange = 5
    
    return cell
}
```

### 卡片预览烟花
```swift
private func setupFireworksEffect() {
    let emitter = CAEmitterLayer()
    emitter.emitterPosition = CGPoint(
        x: containerView.bounds.width / 2,
        y: containerView.bounds.height  // 底部发射
    )
    emitter.emitterShape = .point
    emitter.renderMode = .additive
    
    // 6种彩色粒子
    let colors: [UIColor] = [
        .systemYellow, .systemOrange, .systemRed,
        .systemPink, .systemPurple, .systemBlue
    ]
    
    var cells: [CAEmitterCell] = []
    for color in colors {
        let cell = createMiniFireworkCell(color: color)
        cells.append(cell)
    }
    
    emitter.emitterCells = cells
    containerView.layer.insertSublayer(emitter, at: 0)
    self.emitterLayer = emitter
}
```

### 粒子图像生成
```swift
private func createMiniParticleImage(color: UIColor) -> UIImage {
    let size = CGSize(width: 6, height: 6)
    let renderer = UIGraphicsImageRenderer(size: size)
    
    return renderer.image { context in
        color.setFill()
        let rect = CGRect(origin: .zero, size: size)
        context.cgContext.fillEllipse(in: rect)
        
        // 发光效果
        context.cgContext.setShadow(offset: .zero, blur: 2, color: color.cgColor)
        context.cgContext.fillEllipse(in: rect)
    }
}
```
