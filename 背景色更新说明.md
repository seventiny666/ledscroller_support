# 背景色更新说明

## 更新概述
将应用整体背景色从纯黑色改为深蓝紫色（#0E0F35），营造更具科技感和未来感的视觉氛围。

## 颜色变更

### 新背景色
- **颜色代码**: #0E0F35
- **RGB值**: (14, 15, 53)
- **色调**: 深蓝紫色
- **特点**: 
  - 保持暗色主题
  - 增加蓝紫色调
  - 更具科技感
  - 与青绿色强调色（#8EFFE6）形成互补

### 修改前后对比
| 区域 | 修改前 | 修改后 |
|------|--------|--------|
| 页面背景 | #000000 (纯黑) | #0E0F35 (深蓝紫) |
| 导航栏 | RGB(0.05, 0.05, 0.05) | #0E0F35 |
| TabBar | RGB(0.05, 0.05, 0.05) | #0E0F35 |

## 颜色方案

### 完整配色
| 元素 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #0E0F35 | 页面、导航栏、TabBar |
| 卡片容器 | RGB(0.1, 0.1, 0.1) | 设置卡片、模版卡片 |
| 强调色 | #8EFFE6 | TabBar选中、图标、按钮 |
| 渐变装饰 | #C1FFF4 → #FF6BD6 | 分类标题装饰条 |
| 主文字 | 白色 | 标题、正文 |
| 次要文字 | systemGray | 未选中项、辅助信息 |

### 色彩搭配理念
1. **深蓝紫背景** (#0E0F35)
   - 神秘、科技、未来感
   - 比纯黑更有层次
   - 适合LED/霓虹主题

2. **青绿强调色** (#8EFFE6)
   - 明亮、清新、现代
   - 与深蓝紫形成对比
   - 霓虹灯效果

3. **渐变装饰** (#C1FFF4 → #FF6BD6)
   - 青绿到粉色的过渡
   - 增加视觉趣味
   - 呼应霓虹主题

## 技术实现

### 十六进制转RGB
```swift
// #0E0F35 转换为 RGB
let red = 0x0E / 255.0   // 14 / 255 = 0.0549
let green = 0x0F / 255.0 // 15 / 255 = 0.0588
let blue = 0x35 / 255.0  // 53 / 255 = 0.2078

UIColor(red: 0x0E/255.0, green: 0x0F/255.0, blue: 0x35/255.0, alpha: 1)
```

### 统一应用
所有页面使用相同的背景色代码，确保视觉一致性：
```swift
UIColor(red: 0x0E/255.0, green: 0x0F/255.0, blue: 0x35/255.0, alpha: 1)
```
