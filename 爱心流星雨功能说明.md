# 爱心流星雨功能说明

## 功能概述
在首页新增了一个爱心流星雨卡片（放在第一个位置），点击后进入全屏爱心流星雨动画效果。

## 实现细节

### 1. 新增文件
- `GlowLed/LoveRainViewController.swift` - 爱心流星雨视图控制器

### 2. 爱心流星雨特性
- **黑色背景**：纯黑色背景营造浪漫氛围
- **粉色代码雨**：15列代码雨效果，包含0、1和各种爱心符号（❤、♥、💕、💖、💗、💓）
- **心形轮廓**：中心位置的粉色心形轮廓，带脉动动画
- **"爱"字**：心形中心显示"爱"字，带发光和脉动效果
- **心形粒子**：心形内部持续发射小爱心粒子
- **流星雨**：随机位置发射粉色流星，每秒2个
- **竖屏显示**：全屏竖屏模式

### 3. 动画效果

#### 代码雨效果
- 15列垂直下落的代码字符
- 字符包括：0、1、❤、♥、💕、💖、💗、💓
- 粉色渐变效果
- 持续循环播放

#### 心形轮廓
- 200px大小的心形路径
- 粉色描边，带阴影发光
- 脉动动画：0.95-1.05倍缩放，1.5秒周期

#### "爱"字
- 80pt字体大小
- 粉色带发光效果
- 脉动动画：1.0-1.1倍缩放

#### 心形粒子
- 从心形中心向四周发射
- 小爱心图案
- 旋转、缩放、淡出效果

#### 流星雨
- 每秒2个流星
- 从屏幕上方随机位置发射
- 45度角向右下方飞行
- 粉色渐变拖尾效果
- 1.5秒后自动消失

### 4. 首页预览效果
- 显示心形轮廓
- 中心显示"爱"字
- 周围12个小爱心围成圆形
- 黑色背景

### 5. 数据模型更新
在 `LEDItem.swift` 中添加：
- `isLoveRain: Bool` - 标识是否为爱心流星雨

### 6. 默认卡片
```swift
LEDItem(
    id: "love-rain-special",
    text: "💖 爱心流星雨",
    fontSize: 42,
    textColor: "#FF69B4",
    backgroundColor: "#000000",
    glowIntensity: 4.0,
    scrollType: .none,
    speed: 1.0,
    isLoveRain: true
)
```

### 7. 视觉设计
- **背景色**：纯黑色 (#000000)
- **主题色**：粉色 (systemPink / #FF69B4)
- **代码雨**：粉色字符，0.3倍缩放
- **心形大小**：200px
- **"爱"字大小**：80pt
- **流星速度**：300点/秒
- **粒子发射率**：20个/秒

### 8. 交互逻辑
- 点击首页爱心流星雨卡片 → 进入全屏动画
- 不可编辑、不可删除（特殊卡片）
- 点击"完成"按钮返回首页
- 页面消失时自动停止所有动画

## 技术实现

### CAEmitterLayer 粒子系统
- 代码雨：15个发射器，每列独立
- 心形粒子：圆形发射区域
- 流星：点状发射器，定时创建

### CAShapeLayer 心形路径
- 使用贝塞尔曲线绘制心形
- 添加描边和阴影
- CABasicAnimation 实现脉动

### CADisplayLink 定时器
- 每秒2帧频率
- 创建流星效果
- 自动清理过期粒子

### 自定义图片生成
- `createCodeCharacterImage()` - 代码字符
- `createSmallHeartImage()` - 小爱心
- `createMeteorImage()` - 流星拖尾

## 使用方法

### 在Xcode中添加文件
1. 打开Xcode项目
2. 在左侧项目导航器中，右键点击"GlowLed"文件夹
3. 选择"Add Files to GlowLed..."
4. 找到并选择 `GlowLed/LoveRainViewController.swift`
5. 确保勾选：
   - ✅ Copy items if needed
   - ✅ Create groups
   - ✅ Add to targets: GlowLed
6. 点击"Add"
7. 按 Cmd+B 构建项目

### 测试步骤
1. 运行应用
2. 在首页第一个位置找到"💖 爱心流星雨"卡片
3. 点击卡片进入全屏动画
4. 观察代码雨、心形、"爱"字、粒子和流星效果
5. 点击"完成"按钮返回首页

## 性能优化
- 使用 CAEmitterLayer 硬件加速
- 流星自动清理，避免内存泄漏
- 页面消失时停止所有动画
- 合理的粒子发射率

## 注意事项
1. 爱心流星雨卡片不能编辑或删除
2. 全屏显示时强制竖屏
3. 返回首页时恢复竖屏
4. 所有动画在页面消失时自动停止
5. 流星每1.5秒自动清理
