### 3. 文字输入框重写 ✅
创建了全新的 `createSimpleTextField()` 方法：

```swift
private func createSimpleTextField() -> UITextField {
    let field = UITextField()
    
    // 基础样式
    field.backgroundColor = UIColor(white: 0.2, alpha: 1)
    field.textColor = .white
    field.font = .systemFont(ofSize: 18)
    field.placeholder = "输入LED文字"
    field.layer.cornerRadius = 10
    field.layer.borderWidth = 1
    field.layer.borderColor = UIColor.white.withAlphaComponent(0.2).cgColor
    
    // 内边距
    let paddingView = UIView(frame: CGRect(x: 0, y: 0, width: 15, height: 50))
    field.leftView = paddingView
    field.leftViewMode = .always
    
    // 键盘配置
    field.returnKeyType = .done
    field.enablesReturnKeyAutomatically = true
    field.clearButtonMode = .whileEditing
    field.autocorrectionType = .no
    field.autocapitalizationType = .none
    field.spellCheckingType = .no
    
    // 设置代理和事件
    field.delegate = self
    field.addTarget(self, action: #selector(textFieldDidChange), for: .editingChanged)
    
    field.translatesAutoresizingMaskIntoConstraints = false
    return field
}
```

关键改进：
- 使用更深的背景色 (white: 0.2) 提高对比度
- 添加边框增强视觉效果
- 简化配置，避免复杂的样式冲突
- 使用 `.editingChanged` 事件实时更新预览

## 技术细节

### 字体回退机制
所有字体使用都包含回退：
```swift
UIFont(name: fontName, size: fontSize) ?? .boldSystemFont(ofSize: fontSize)
```

如果系统字体不可用，会自动使用系统粗体作为备选。

### 预览实时更新
通过以下方式实现实时预览：
- `textFieldDidChange` 监听文字输入
- `fontSizeChanged` 监听字体大小
- `glowChanged` 监听霓虹强度
- `fontChanged` 监听字体选择
- `colorButtonTapped` 监听颜色选择

所有改变都会调用 `updatePreview()` 方法更新预览区。
